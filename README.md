# Sales-Performance-Dashboard

# Overview
An Interactive Sales Performance Dashboard with responsive UI design. The dashboard transforms sales data into actionable insights through executive KPIs, dynamic filtering, multi-tab navigation, and interactive visualizations covering sales, profitability, customer performance, and operations.

# Features
- Executive overview with real-time KPI cards.
- Multi-tab dashboard (Executive, Sales, Customer, Operations).
- Interactive multi-select filters with instant cross-filtering.
- Dynamic charts using Chart.js (line, bar, stacked bar, horizontal bar, and doughnut).
- Sales, profit, margin, and operational performance analysis.
- Customer segment and sales representative performance tracking.
- Order status, delivery, and operational efficiency insights.
- Responsive and modern dashboard UI with reusable components.
- Automatic KPI recalculation based on applied filters.

# Skills Demonstrated
- Frontend Dashboard Development.
- Interactive Data Visualization.
- Responsive Web Design.
- Dynamic Filtering.
- KPI Design & Business Analytics.
- Data Aggregation & Transformation.
